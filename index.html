Getting Started

아이디필터 관리자페이지 UI 가이드

아이디필터 관리자페이지의 어쩌구 저쩌구 왜 만들게됐나 글로 잘 설명하기가 어렵네 한번 잘~ 만들어 놓은 제품을 계속 기능추가, 개선 등등 하면서 개발시 추가되는 컴포넌트 등의 톤앤매너 유지에 도움을 주기위해 만들었따??

  • 해당 페이지는 UI는 부트스트랩4 기반으로 제작 되었습니다.
  • sass 코딩후 css 컴파일하여 제작되었기에 기존의 외부 css파일 수정은 불가합니다. 스타일 수정, 추가가 필요할 경우 내부 스타일 시트 또는 인라인으로 작성 합니다. (단, 인라인으로 작성시 추후 유지보수, 관리에 어려움이 있을 수 있으니 가급적 자제 바랍니다.)
  • 또 무슨말을 하면 좋을까 알려주세요 할말이 많은거 같긴한데 뭐라고 써야할지 모르겠따아아아아

Colors

아이디필터의 메인컬러인 민트 컬러를 기본으로 그에 어울리는 컬러를 사용하여 디자인하였습니다. 컬러 사용의 자세한 방법은 각 항목별로 확인 바랍니다.

Primary#06c1b7

아이디필터의 메인컬러로 주요 확인, 저장 버튼 또는 텍스트, 배경에 강조 컬러로 사용합니다.

Secondary#abb3b7

아이디필터의 서브컬러로 주로 Primary컬러와 함께 조합하여 취소, 삭제 버튼 등에 사용합니다.

Info#3869e0

메인컬러 다음으로 포인트되는 컬러로 버튼, 텍스트, 배경에 강조 컬러로 사용합니다.

Danger#f44545

주의, 경고, 반려와 같은 의미로 쓰이는 버튼, 뱃지, 텍스트에 사용합니다.

Light#f8f9fa

table의 th,td 또는 박스안에 음영으로 사용하는 컬러입니다.

Navy#1d385e

페이지의 다크버전에서 사용하는 컬러로 관리자페이지에서는 그래프에 사용됩니다.

Fonts

  • 폰트는 대표적인 국문 웹폰트인 Noto Sans(본고딕)을 사용합니다. 구글 웹폰트에서 다운로드 받아 사용할 수 있습니다.
  • fonts/noto 폴더 안에서 woff,woff2 포멧을 제공합니다.
  • 폰트의 두께는 Regular(400), Bold(600)로 사용합니다.

사용 예시

              

                <p class="font-weight-bold">Noto Sans 볼드 타입</p>
                <p class="font-weight-normal">Noto Sans 레귤러 타입</p>

Noto Sans 볼드 타입

Noto Sans 레귤러 타입

Icons

  • 아이디필터 초기 버전에서는 .svg파일의 아이콘을 적용하였으나, 유지보수 또는 추가관리에 어려움이 있어 결합관리 버전에서는 부트스트랩에서 제공하는 아이콘을 사용하였습니다.
  • 부트스트랩 아이콘 페이지를 참고하여 적용합니다.
  • 버튼에 텍스트와 함께 사용시, 아이콘과 텍스트 사이의 여백을 꼭 넣어주세요.
  • 자주 사용하는 대표 아이콘을 확인하세요.

Layout

  • 페이지의 레이아웃은 좌측 네비게이션, 우측 컨텐츠 영역으로 나뉩니다.
  • 우측 컨텐츠 영역의 너비는 최대 100%, 최소 1180px로 정의 하였으며 임의 수정은 불가합니다.
  • 컨텐츠 영역은 1단, 2단 또는 2단과 1단이 한페이지에 존재 할 수 있습니다.
